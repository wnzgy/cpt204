# Report Assembly Guide

Use this order to assemble the final Word report:

1. Cover page (module template)
2. Chapter 1-3 text from `docs/report_ch1_ch3_draft.md`
3. Chapter 4 text from `docs/report_ch4_reflection_draft.md`
4. Chapter 5 code from `docs/chapter5_program_code.md`
5. Chapter 6 appendix from `docs/chapter6_appendix.md`
6. Chapter 7 contribution table from `docs/chapter7_contribution_form.md`

Final pass:

- Apply `docs/word_format_checklist.md`
- Verify numbers against `outputs/*.csv`
- Confirm all placeholders are replaced with real group information
