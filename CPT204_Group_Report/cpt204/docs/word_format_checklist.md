# Word Format Checklist

Use this checklist before final export/submission.

## Document format

- Font: `Calibri`
- Font size: `12`
- Line spacing: `1.5`
- Margins: `Normal`

## Structure check

- Cover page included
- Chapter 1 (Task A) completed
- Chapter 2 (Task B) completed
- Chapter 3 (Task C) completed
- Chapter 4 (Task D) completed
- Chapter 5 (Program code as text) completed
- Chapter 6 (Appendix) completed
- Chapter 7 (Contribution form) completed

## Content check

- Main body page limit checked (excluding cover and Chapters 5-7)
- All tables/outputs match files in `outputs/`
- Graph case values match `outputs/graph_cases.csv`
- AI citation statement added in appendix
- Contribution percentages sum to `100%`
