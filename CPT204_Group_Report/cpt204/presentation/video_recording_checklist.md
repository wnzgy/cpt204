# Video Recording Checklist (Task E)

## Before recording

- [ ] Finalize slides using `presentation/CPT204_Project_Presentation.md`
- [ ] Rehearse using `presentation/speaker_notes.md`
- [ ] Confirm both members appear on camera
- [ ] Confirm both members use own voice
- [ ] Confirm total duration <= 8 minutes

## During recording

- [ ] Follow the planned timeline
- [ ] Keep transitions clear between two speakers
- [ ] Ensure all required tasks are covered:
  - [ ] Project intro
  - [ ] Task A sorting
  - [ ] Task B graph
  - [ ] Task C OOP design
  - [ ] Task D reflection

## After recording

- [ ] Export MP4
- [ ] Check duration, audio, and face visibility
- [ ] Save final PPT used in recording
- [ ] Ensure PPT and MP4 versions are consistent
